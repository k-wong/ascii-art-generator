## Features

- Convert image files to ASCII text.
- Tune width, height, contrast, brightness, gamma, edge boost, character set, and resize kernel.
- Use as a command-line tool or import as a Python library.
- Optionally export browser-ready JavaScript globals for static sites.
- Optionally sample videos into ASCII frames with the `video` extra.

## Installation

After publishing to PyPI:

```bash
python -m pip install ascii-art-generator
```

For local development from this repository, or before the package is published:

```bash
python -m pip install -e ".[dev]"
```

For video support:

```bash
python -m pip install "ascii-art-generator[video]"
```

For local development with video support:

```bash
python -m pip install -e ".[video]"
```

## CLI Usage

```bash
ascii-art input.jpg --output output.txt --width 160
```

Use a custom character set by passing characters from darkest to lightest:

```bash
ascii-art input.jpg --width 120 --charset "@#*+=-. " --print
```

Print directly to the terminal:

```bash
ascii-art input.jpg --width 120 --print
```

Generate a JavaScript file for a static website:

```bash
ascii-art input.jpg --output-js ascii.js --js-variable HERO_ASCII
```

Generate ASCII frames from a video:

```bash
ascii-art-video input.mp4 --output-dir frames --fps 8 --frames 120 --width 120
```

Generate text frames plus browser-ready JavaScript frame data:

```bash
ascii-art-video input.mp4 \
  --output-dir frames \
  --output-js video_frames.js \
  --js-variable HERO_ASCII_FRAMES
```

## Python Usage

```python
from ascii_art_generator import AsciiOptions, image_to_ascii

ascii_art = image_to_ascii(
    "input.jpg",
    options=AsciiOptions(width=120, charset="@%#*+=-:. "),
)

print(ascii_art)
```

You can also pass a charset string as a keyword override when you do not need a full
`AsciiOptions` object:

```python
from ascii_art_generator import image_to_ascii

ascii_art = image_to_ascii("input.jpg", width=120, charset="01")
```

If you already have a Pillow image:

```python
from PIL import Image
from ascii_art_generator import image_to_ascii_from_image

image = Image.open("input.jpg")
ascii_art = image_to_ascii_from_image(image, width=120)
```

## Notes For Maintainers

- Package code lives in `src/ascii_art_generator`.
- The browser JavaScript export defaults to `window.ASCII_ART`; pass `--js-variable` for a project-specific name.
- Video export defaults to `window.ASCII_VIDEO_FRAMES`; pass `--js-variable` for a project-specific name.
